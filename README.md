# QI_servicosWeb
Programa apresentado no trabalho de web services da disciplina de web services qi:
	*Desenvolvido em Spring-boot
	*Cria, um Arraylist de 10 elementos de objetos com alguns atributos.
	*A aplicação desponibiliza 3 endpoints, que apresenta todos elementos, apenas 1 elemento e o ultimo deleta um elemento.
	*O front end com faz uso de css e javascript
